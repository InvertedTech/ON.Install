# ts-node-grpc
Just a quick example of a gRPC server made with typescript and node js! [Video link](https://youtu.be/0cxEVcALoxc)
It contains 4 different examples:
1. Unary
2. Server Streaming
3. Client Streaming
4. Bidirectional Streaming


## Project setup
```
yarn install
```

### Runs the gRPC server
```
yarn start
```

### Runs out client
```
yarn client
```
